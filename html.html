HTML

HyperText Markup Language is not a programing language like C++ or Java. It is a markup language. A markup language is a set of instructions on how to create a document like a webpage. The current standard is HTML5. It is the first substantial update since 1999.

Elements

"HyperText" refers to links that connect web pages to one another, either within a single website or between websites. HTML uses "markup" to annotate text, images, and other content for display in a Web browser. HTML markup includes special "elements" such as...

  1. <head>
  2. <title>
  3. <body>
  4. <div>
  5. <li>
  6. and many more

These elements help give your site structure for both the site itself and anyone who might be reading/adjusting your code. Lets dive into each of these elements...

<head>

The <head> element is a container for metadata (or data about data) and it is placed between the <html> and the <body> tag. HTML metadata is data about the HTML document. Metadata is not displayed. Metadata typically define the document title, character set, style, scripts, and other meta information

<title>

The <title> element defines the title of the document. The title must be text-only, and it is sown in the browser's title bar or in the page's tab. The content of a page title is very important for search engine optimization (SEO)! The page title is used by serach engine algorithms to decide the order when listing pages in search results.

<body>

The <body> tag defines the document's body. The <body> element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. Note: There can only be one <body> element in an HTML document.

<div>

<div> (where div is short for 'division') is used to define a particular portion of a page.The <div> element is by defailt a block element, meaning that it takes all available width, and comes with line breaks before and after. The <div> is usually given a class or id so that you are able to stlye or adjust it from other languages (i.e. CSS adding color/styling and JavaScript for animations/funcationality.

<body>

The <body> tag defines the document's body. The <body> element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc. Note: There can only be one <body> element in an HTML document.

<li>

The <li> ('meaning List Item') element is used to represent an item in a list. It must be contained in a parent element: an ordered list (<ol>), an unordered list (<ul>) or a menu (<menu>). In menus and unordered lists, list items are usually displayed using bullet points. In ordered lists, they are usually displayed with an ascending counter on the left, such as a number or letter.

<a>

The <a> element is used to create hyperlinks in an HTML document. They hyperlinks can point to other webpages, file on the same server, a location on the same page, or any other URL via the hyperlink reference attribute,href. The href determines the location the anchor element points to.

<video>

The <video> element embeds a media player for video playback. The src attribute will contain the url to the video. Adding the controls attribute will display vdieo controls in the media player. Note: The content inside the opening and closing tag is shown as a fallback in browsers that don't support the element.

<em>/<strong>

There are a multitude of ways to style and adjust text through HTML. Two of the more basic in-line element adjustors would be <em> and <strong>.
-The <em> element emphasizes text and browsers will usually italicize the emphasized text by default.
-The <strong> element highlights important, serious, or urgent text and browsers will normally render this highlighted text in bold by default.

Tables

HTML tables allow web developers to arrange data into rows and columns. To set up a table in HTML, there are several tags that are implemented.
